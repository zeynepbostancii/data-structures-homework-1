1. Hafta Ödevi – Array İşlemleri

Konu
Array (Dizi) üzerinde temel işlemler yapılacak ve GitHub’a yüklenecek.

Yapılan İşlemler

- Traversal (Gezme)
  - Dizinin tüm elemanlarını sırayla dolaşma
- Insertion (Ekleme)
  - Belirli bir index’e yeni eleman ekleme
- Deletion (Silme)
  - Belirli bir index’teki elemanı silme
- Reverse (Tersine Çevirme)
  - Diziyi baştan sona ters çevirme

Notlar
- Ekleme ve silme işlemlerinde kaydırma (shift) yapılmıştır
- Kod C dili ile yazılmıştır
-Bu ödevde dinamik bellek kullanımı yapılmamış, boyutu sabit (statik) array yapısı tercih edilmiştir.
